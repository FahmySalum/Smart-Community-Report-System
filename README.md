# Smart Community Reporting System

Web Application

## Overview
This project is designed to help community members report public issues such as:
- Road damage
- Water supply problems
- Electricity faults
- Illegal dumping
- Security concerns
- Environmental issues

The system enables authorities to review reports, update their status, and provide feedback to citizens.

## Features

- User Registration
- User Login
- Report Community Issues
- Upload Images
- Track Report Status
- Receive Feedback
- Admin Dashboard
- Manage Users
- Manage Reports
- Generate Reports
